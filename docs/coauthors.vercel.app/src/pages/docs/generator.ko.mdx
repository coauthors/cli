# Coauthors 생성기

<br/>

<CoauthorsGenerator/>

<Callout type='info'>
함께한 동료를 GitHub [co-author로 추가](https://docs.github.com/en/pull-requests/committing-changes-to-your-project/creating-and-editing-commits/creating-a-commit-with-multiple-authors#creating-co-authored-commits-on-the-command-line)하세요. 

당신의 커밋을 돕는 사람의 노고를 인정하기 위한 좋은 방식입니다.

```shell
$ git commit -m "Refactor usability tests.
>
>
Co-authored-by: NAME <NAME@EXAMPLE.COM>
Co-authored-by: ANOTHER-NAME <ANOTHER-NAME@EXAMPLE.COM>"
```
</Callout>
