# VSCode Extension

We provide a simple VSCode extension that allows you to enter a GitHub username and create a commit message.

## How to use

1. Install the extension from the [VSCode Marketplace](https://marketplace.visualstudio.com/items?itemName=coauthors.coauthors-vscode).
2. Open the command palette (Ctrl+Shift+P) and type `Coauthors: Add Coauthor`.
3. Enter the GitHub username of the co-author you want to add.
4. The extension will automatically generate a commit message with the co-author added.
